Fichiers sources de la page d'accueil interactive pour le site vitrine WordPress.
Format Point & Click avec une énigme à résoudre pour accéder au site vitrine.
Le mot de passe est 1413.

Le site WordPress est livré séparément dans une archive contenant le site et la base de données.

Ce dernier devra être installé à la racine du projet ci-présent.

La carte umap est jointe à ce projet. Pour l'éditer, il faut avoir un compte OpenstreetMap et y importer le fichier .umap.
